# Contributing

Thank you for contributing! Follow these steps for a smooth contribution flow:

1. Fork the repo and create a branch: `feature/<short-description>`
2. Keep changes focused and add tests where applicable
3. Run the pre-commit checks locally:

```bash
git config core.hooksPath .githooks
```

4. Sign your commits: `git commit -S -m "..."`
5. Push your branch and open a Pull Request against `main`

Code style and tests may be enforced by repository hooks; follow any feedback on CI.
